# login EC2 instance

```bash
ssh -i your-key.pem ubuntu@your-ec2-ip
```

# create Makefile

```bash
nano Makefile
```

# install All:

```bash
make all
```

# specific install :

```bash
make install-git
```

## Docker install

```bash
make install-docker
```

## Node.js & npm install

```bash
make install-node
```
